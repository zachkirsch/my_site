Boggle

I created the C++ backend of a Boggle game as a capstone project for my Data Structures course. The program features a 2D dynamic array for the board, a radix tree for the dictionary (to enable fast searching), and a recursive algorithm for solving the board. Source code is available upon request.

  • Class: Data Structures
  • Date: April 2015
  • Area: C++ Backend Development

Nickel and Dine

I worked on a team of four to create Nickel and Dine for my Web Programming course. Nickel and Dine is a food delivery service for Tufts students to generate orders that meet each restaurant's delivery minimum and show the final cost of the order (tax, tip, and fees included). We created the server using Node.js, kept track of orders using a MongoDB database, and built the frontend with the help of Bootstrap. Source code is available upon request.

  • Class: Web Programming
  • Date: April 2015
  • Area: Web Frontend & Backend Development

DebateHelper

DebateHelper is a Microsoft Word Template for high school debaters that I created to solve issues with organization and collaboration within my debate team. I used VBA 7 and XML to design a macro system that makes it easier to keep track of large amounts of information and stay organized. The template is now used by debate teams across the country. Source code is available upon request.

  • Date: Summer 2014
  • Area: VBA 7
