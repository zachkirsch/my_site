Boggle

I created the C++ backend of a Boggle game as a capstone project for a data structures course. The program features a 2D dynamic array for the board, a radix tree for the dictionary to optimize searching, and a recursive algorithm for solving the board. Source code is available upon request.

  • Class: Data Structures
  • Date: April 2015
  • Area: C++ Backend Development

Nickel and Dine

I worked on a team of four to create Nickel and Dine for a web programming course. Nickel and Dine is a food delivery service for Tufts students to generate orders that meet each restaurant's delivery minimum and show the final cost of the order (tax, tip, and fees included). We created the server using Node.js, kept track of orders using a MongoDB database, and built the frontend with the help of Bootstrap. Source code is available upon request.

  • Class: Web Programming
  • Date: April 2015
  • Area: Web Frontend & Backend Development

DebateHelper

I created DebateHelper to solve issues with organization and collaboration within my debate team. I used VBA 7 and XML to design a macro system in Microsoft Word that makes it easier to keep track of large amounts of information and stay organized. The template is now used by over a thousand debaters across the country. Source code is available.

  • Date: Summer 2014
  • Area: VBA 7

Engineers Without Borders

At Tufts, I'm the Co-Treasurer of Engineers Without Borders, a club that plans and executes sustainable engineering projects in developing countries. The current project we are working on is a water pumping system in Shilongo, Uganda that pumps during the night so that villagers can have easy access to water.

This summer, I built a website for the group in order to practice web development and design. I've really enjoy working on the projects and being a part of this great organization.

  • Date: Summer 2015
  • Area: Web Development

Victory Briefs

Victory Briefs is a company focused on high school debate. They provide research and evidence on debate topics and host camps every summer for hundreds of debaters.

I began working for Victory Briefs in October 2014, when the company purchased an Elo rankings system that I developed. The system includes an Elo-based methodology and a Java backend for inputting thousands of pairings and determining the ranking of each team. It is now used for their national ranking of high school debaters.

I also built a lecture signup system for the summer debate camp using Node.js, MongoDB, and the Google Apps API.

In August 2015, I became the Head of Web Development for Victory Briefs. I built their website and am currently working on improving their blog.

  • Date: October 2014 - Present
  • Area: Web Development
